Step 1: Supabase Setup (5 min)

  1. Go to supabase.com ‚Üí Create free account
  2. Create new project (any region works)
  3. Go to Settings > API
  4. Copy these 3 keys to a safe document:
    • Project URL ‚Üí Save as PUBLIC_SUPABASE_URL
    • anon key ‚Üí Save as PUBLIC_SUPABASE_ANON_KEY
    • service_role key ‚Üí Save as SUPABASE_SERVICE_ROLE_KEY
  5. Go to SQL Editor in Supabase
  6. Create new query and paste the SQL from SUPABASE_SETUP.sql
  7. Click RUN - this creates your database tables
‚úì Checkpoint: You should see 4 new tables in Supabase: profiles, content, soul_maps, subscriptions

Step 2: Stripe Setup (5 min)

  1. Go to stripe.com ‚Üí Create account
  2. Go to Developers > API Keys
  3. Copy these 2 keys:
    • Publishable key (starts with pk_) ‚Üí Save as PUBLIC_STRIPE_PUBLIC_KEY
    • Secret key (starts with sk_) ‚Üí Save as STRIPE_SECRET_KEY
  4. Create 3 Price objects:
    • Click Products > Create product
    • Name: "Ignite", Price: $33/month, recurring ‚Üí Copy price ID (price_xxxxx)
    • Name: "Soul Map", Price: $111/month, recurring ‚Üí Copy price ID
    • Name: "The First Spark", Price: $1000, one-time ‚Üí Copy price ID
  5. Update these IDs in file: src/lib/tiers.js
    stripe_price_id: 'price_1234567890' ‚Üê Replace with real ID
‚úì Checkpoint: You have 3 price IDs from Stripe and updated tiers.js

Step 3: Deploy to Vercel (5 min)

  1. Push your code to GitHub (create repo if needed)
  2. Go to vercel.com ‚Üí Sign up / Log in
  3. Click Import Project
  4. Select your GitHub repo
  5. Go to Environment Variables and add these 5:
    • PUBLIC_SUPABASE_URL
    • PUBLIC_SUPABASE_ANON_KEY
    • PUBLIC_STRIPE_PUBLIC_KEY
    • STRIPE_SECRET_KEY
    • SUPABASE_SERVICE_ROLE_KEY
  6. Click Deploy
  7. Wait 2-3 minutes for build to complete
‚úì Checkpoint: Your site is LIVE! Vercel gives you a URL like first-spark-hub-xxxxx.vercel.app

Step 4: Test Everything (3 min)

  1. Visit your live URL
  2. Click Sign Up and create test account
  3. Go to Pricing and click a tier
  4. Use Stripe test card: 4242 4242 4242 4242 (any future date, any CVC)
  5. Check your dashboard loaded
‚úì You're LIVE! The First Spark membership hub is running
