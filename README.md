# ✦ THE FIRST SPARK ✦
## Cosmic Reciprocity Engine

An **out-of-this-world** presentation deck showcasing The First Spark consciousness platform concept with stunning cosmic branding and futuristic design.

---

## 🌌 Cosmic Features

### Visual Design
- **Starfield Background**: 200 twinkling stars creating an immersive space atmosphere
- **Nebula Gradients**: Dynamic, color-shifting backgrounds on each slide
- **Cosmic Pulses**: Animated glowing effects that breathe life into the presentation
- **Particle Effects**: Interactive particles that follow your cursor over key elements

### Typography & Branding
- **Orbitron Font**: Futuristic, space-age headings with letter-spacing
- **Space Grotesk Font**: Clean, modern body text
- **Glowing Text**: All key highlights feature neon glow effects in:
  - **Cyan** (#00d9ff) - Future/Technology
  - **Magenta** (#ff1f7a) - Warning/Extraction
  - **Green** (#00ff88) - Success/Reciprocity
  - **Purple** (#c084fc) - Innovation/Engine

### Interactive Elements
- **Glassmorphic Cards**: Frosted glass effect with backdrop blur
- **Hover Animations**: Elements lift and glow on hover
- **Rotating Gradients**: Subtle rotation effects on engine cells
- **Shimmer Progress Bar**: Animated gradient progress indicator
- **Liquid Button Effects**: Expanding ripple on button hover

### Animations
- **Slide Transitions**: Smooth cubic-bezier entrance with blur
- **Brand Mark**: Pulsing cosmic logo with spinning star icon
- **Shine Effects**: Moving light reflections on CTA buttons
- **Particle Trails**: Floating particles on interactive elements
